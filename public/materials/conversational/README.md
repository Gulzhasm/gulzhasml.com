# Conversational Agents — Lecture PDFs

Place your course lecture PDFs here so they are linked from the Learn → Conversational Agents pages.

**Expected filenames** (used by the app):

- `week1-foundations.pdf` — Foundations of Dialogue Systems
- `week2-dialogue-act-tagging.pdf` — Dialogue Act Tagging
- `week3-seq2seq-dialogue.pdf` — Seq2Seq for Dialogue
- `week4-rag-dialogue.pdf` — RAG for Dialogue
- `week5-task-oriented-agent.pdf` — Task-Oriented Agent (Restaurant Booking)

URLs will be `/materials/conversational/weekN-….pdf`. If your files use different names, update the `lecturePdfUrl` values in `src/lib/topics.ts` for the conversational sections.
