# Deep Work + Chimp Paradox: Your Performance System

## Part 1: Cal Newport's Deep Work Principles

### The Core Idea
Deep work = Professional activities performed in a state of distraction-free concentration that push your cognitive capabilities to their limit.

### Rules to Live By

#### Rule 1: Work Deeply
- **Ritualize**: Same time, same place, same routine
- **Set constraints**: 60-90 min blocks (matches ultradian rhythm)
- **Shutdown ritual**: Clear end to work mode ("Shutdown complete")

#### Rule 2: Embrace Boredom
- Don't reach for phone during idle moments
- Schedule internet blocks, not breaks from internet
- Practice being bored - it builds concentration muscle

#### Rule 3: Quit Social Media (or heavily restrict)
- Apply the craftsman approach: Does this tool provide substantial value to my goals?
- 30-day detox experiment for questionable tools

#### Rule 4: Drain the Shallows
- Schedule every minute of your day (time blocking)
- Quantify depth of each task
- Ask: "How long would it take to train a smart grad to do this?"

### Deep Work Scheduling Strategies

| Strategy | Description | Good For |
|----------|-------------|----------|
| Monastic | Eliminate all shallow work | Writers, researchers |
| Bimodal | Dedicated deep seasons/days | Academics |
| Rhythmic | Daily habit, same time | **You - 60-90 min morning block** |
| Journalistic | Fit it in when you can | Experienced practitioners |

## Part 2: The Chimp Paradox Techniques

### The Mind Model
Your brain has 3 systems:
1. **Chimp** (Limbic) - Emotional, reactive, 5x faster than Human
2. **Human** (Frontal) - Logical, rational, your true self
3. **Computer** (Stored patterns) - Autopilot behaviors, beliefs

### Key Insight
You cannot control your Chimp, but you can **manage** it.

### Managing Your Chimp

#### 1. Exercise the Chimp (Don't suppress!)
- Let it vent (journal, talk to yourself, walk)
- Acknowledge emotions: "I notice I'm feeling anxious about this"
- Give it 10 minutes to rage, then engage Human

#### 2. Box the Chimp
- Feed it what it needs: security, recognition, purpose
- Create "Truths" (facts it must accept)
- Build "Autopilots" (automatic good behaviors)

#### 3. Feed Your Computer Good Programs
- **Autopilots**: "When I sit at my desk at 9am, I start deep work"
- **Truths**: "Distraction feels good but leaves me empty"
- **Remove Gremlins**: Old beliefs that sabotage you

### Before Deep Work: Chimp Check
Ask yourself:
1. Is my Chimp calm? (If not, exercise it first - 5 min journal)
2. What is my Chimp worried about? (Address it or park it)
3. What does my Human want to achieve?

## Part 3: Your Combined Daily System

### Morning Routine (Before Deep Work)

```
1. Wake up - no phone for first 30 min
2. Chimp check: Journal 5 min if needed
3. Review today's ONE deep work goal
4. Set environment: phone away, tools ready
5. Start timer, begin work
```

### During Deep Work

```
- Distraction urge? Note it, return to work
- Chimp panicking? "I'll handle that after the session"
- Stuck? Take 2 min walk, return
- Flow state? Extend to 90 min max
```

### End of Session

```
1. Shutdown ritual: Review what's done, capture next action
2. Say "Shutdown complete" (trains your brain)
3. Reward yourself (coffee, walk, music)
4. No work thoughts until next session
```

## Part 4: Weekly Review Questions

1. How many deep work hours did I log?
2. What was my average focus quality (1-5)?
3. When did my Chimp hijack me? What triggered it?
4. What autopilots served me well?
5. What's next week's deep work priority?

## Quick Reference: When Things Go Wrong

| Problem | Deep Work Fix | Chimp Fix |
|---------|--------------|-----------|
| Can't start | Shrink the task to 25 min | Exercise Chimp first (journal) |
| Keep checking phone | Phone in another room | Ask: What is Chimp seeking? |
| Overwhelmed | Pick ONE task only | Write fears down, box them |
| Procrastinating | Schedule specific time block | Chimp fears failure - acknowledge it |
| Low energy | Move session to peak hours | Check: sleep, food, exercise |
